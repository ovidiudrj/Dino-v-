# Dino(v) :t-rex:

A Chrome Dino clone I built to learn game development fundamentals with Python and Pygame.

<p align="center">
  <img src="assets/demo.gif" alt="Gameplay">
</p>

## Controls

- **Space** — jump
- **S** — duck (or fast-fall while in the air)
- **R** — restart after game over

## Running it

You'll need Python and Pygame: pip install pygame + python main.py
